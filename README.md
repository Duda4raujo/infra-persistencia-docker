# Infraestrutura de Persistência com Docker

## Aluna
Maria Eduarda de Araújo Silva

## Objetivo
Esse trabalho tem como objetivo demonstrar mecanismos de persistência de dados em ambientes conteinerizados utilizando Docker, aplicando conceito de volumes, bind mounts, backup, restauração e automação básica de infraestrutura.
